SIDS

Data provided "as is," no warranties

Description

Sudden Infant Death Syndrome sample data for North Carolina counties, two time periods (1974-78 and 1979-84)

Type = polygon shape file, unprojected, lat-lon

Observations = 100

Variables = 14

Source

Cressie, Noel (1993). Statistics for Spatial Data. New York, Wiley, pp. 386-389.

Variables

Variable

Description
