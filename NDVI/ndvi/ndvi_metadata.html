Normalized Difference Vegetation Index Grid

Data provided "as is," no warranties.


File Description
Original data: A subset of 7 by 7 square raster grids with 10 arcminute spacing roughly situated around the border between the Central African Republic, Sudan and Congo, drawn from the NOAA 1990 Global Change Database.

ndvi.dbf:

Data file to join with base grid map

grid77.shp shx dbf:

Base grid map 7 x 7

ndvigrid.shp shx dbf:

Grid with data
