Description

July 1996 ozone measurements for Los Angeles basin monitoring stations.

Type = dat file (input format required by Variowin), projected, UTM

Observations = 32

Variables = 8

Source

Original data provided by James Murdoch, School of Social Sciences, University of Texas at Dallas, Richardson, TX 75083 murdoch@utdallas.edu.
