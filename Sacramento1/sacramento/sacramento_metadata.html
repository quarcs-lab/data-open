2000 Census Tract Data for Sacramento MSA

Data provided "as is," no warranties.


Description
Selected 2000 Employment and Demographic Census Tract Data (Summary File 3) for Sacramento MSA (OMB's 2003 MSA Definition).

Type = Polygon shape file (ArcView)
Observations = 403 Census Tracts
Variables = 17

Source
US Census Bureau, 2000 Census (Summary File 3). Extracted from http://factfinder.census.gov in April 2004.

Notes
Note that the poverty variable names can be easily confused: pov_pop is the reference population (i.e. total population); pov_tot is all poor persons.
