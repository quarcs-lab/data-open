Ozone Data 1997-1999

Data provided "as is," no warranties.


File Description
Monthly ozone measurements for 30 monitoring stations.

oz9799.txt:

Comma delimited text file for input as point file

oz9799.shp shx dbf:

Point shape file

Type = Point shape file
Observations = 30
Variables = 78 (3 years; 72 ozone variables plus 6 identifiers)
Missing Values: -1000

Source
US Environmental Protection Agency

Variable

Description

STATION 

Station ID 

MONITOR 

The monitor to which the data applies 

LATITUDE 

Latitude of monitoring site (UTM Zone 11) 

LONGITUDE 

Longitude of monitoring site (UTM Zone 11) 

X_COORD 

X Coordinates 

Y_COORD 

Y Coordinates 


3 years (1997-99) x 12 months x 2 (max & average):

Mxxy 

Daily maximum for year xx and month y 

Axxy 

Average or highest 8 hour period for year xx and month y
